  "PC" -> ordinateur
  "rame" -> ??? pagaie de bateau ???
  "TTS" -> ???
  -> Resultat : confusion totale

Mila-Assist (comprehension contextuelle) :
  "PC" + "rame" + "TTS" dans le meme contexte
